# Maze-Solver-with-UI
Visualization of searching algorithms like BFS, DFS, Greedy and UCS in the form of maze solver.
## How to Run: ##
* Open the code in Pycharm
* Install following pacakges
    * tkinter
    * Canvas
    * bottom
    * matplotlib
* click on run to run the program.
